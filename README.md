<p>O projeto consiste em uma atividade multidisciplinar que mistura as habilidade de programação web, com programção backend na integração com banco de dados, juntamente com isso, montar um dashboard no Power BI para mostrar os dados cadastrados na aplicação. A aplicação basicamente faz o CRUD de dados convencionais, como o nome de um produto qualquer, registro de um pedido ou até mesmo o cnpj de algum fornecedor.</p>
